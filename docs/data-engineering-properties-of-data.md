# Properties of data

- Volume
- Velocity
- Variety

## Volume

Amount or size of data that is being generated and stored. Can range from gigabytes to petabytes and beyond.

Example:

- Social media platform

## Velocity

Speed at which data is generated, processed, and analyzed. Can be real-time or batch processing.

Example:

- Stock market data feeds
- Sensor data

## Variety

Different types and formats of data, including [structured](./data-engineering-types-of-data.md#structured-data), [semi-structured](./data-engineering-types-of-data.md#semi-structured-data), and [unstructured](./data-engineering-types-of-data.md#unstructured-data) data.

Example:

- Text, images, videos, audio files
- Sensor data
- Log files
