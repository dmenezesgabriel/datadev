# AWS Services

## Amazon Comprehend

### Use Cases

- Find meaningful insights and relationships in text
- Generate summaries of text documents

### Features

- Natural language processing (NLP)
- Entity recognition
- Extract key phrases, entities, events, personally indentifiable information (PII), sentiment(positive, neutral, negative, or mixed), and language from text

## Amazon Kendra

### Use Cases

- Search across unstructured data using natural language queries

### Features

- Retrieval Augmented Generation (RAG)

## Amazon Textract

### Features

- Extracts printed text, handwriting, and other data from scanned documents using machine learning
- Support tabular data

## Amazon Rekognition

### Use Cases

- Content moderation

### Features

- Detect objects, scenes, and faces in images and videos

## Amazon Transcribe

### Features

- Converts video/audio into text

## Amazon Translate

### Features

- Translate text into different languages

## Amazon Augmented AI (Amazon A2I)

### Use Cases

- Human review workflows for machine learning predictions
