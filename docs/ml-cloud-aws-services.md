# AWS Services

## Amazon Comprehend

### Use Cases

- Find meaningful insights and relationships in text
- Generate summaries of text documents

### Features

- Natural language processing (NLP)
- Entity recognition
- Extract key phrases, entities, events, personally indentifiable information (PII), sentiment(positive, neutral, negative, or mixed), and language from text

## Amazon Kendra

### Use Cases

- Search across unstructured data using natural language queries

### Features

- Retrieval Augmented Generation (RAG)

## Amazon Textract

### Features

- Extracts printed text, handwriting, and other data from scanned documents using machine learning
- Support tabular data

## Amazon Rekognition

### Use Cases

- Content moderation

### Features

- Detect objects, scenes, and faces in images and videos

## Amazon Transcribe

### Features

- Converts video/audio into text

## Amazon Translate

### Features

- Translate text into different languages

## Amazon Augmented AI (Amazon A2I)

### Use Cases

- Human review workflows for machine learning predictions

## AWS Trainium (EC2 Hardware)

### Use cases

- Deep learning training of 100B+ parameter models

### Features

- Trn1 instances deliver high performance and low cost for deep learning training in the cloud

## AWS Inferentia (EC2 Hardware)

### Use Cases

- High performance at lowest costs EC2 hardware for deep learning (DL) and generative AI inference applications

### Features

- Inf1 instances deliver 2.3x higher throughput and 70% lower cost per inference compared to Amazon EC2 instances
